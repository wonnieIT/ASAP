# ASAP
Stream live, podcast later, and share game videos – instantly with ASAP (AI Streamer As Podcast)!



## 실행 방법 
1. conda env 생성 
```
conda create --name myenv python=3.9
```

2. conda env 활성화 
```
conda activate myenv
```

3. 설치
```
pip install -r requirements.txt
```

3. 실행
```
python app.py

```
